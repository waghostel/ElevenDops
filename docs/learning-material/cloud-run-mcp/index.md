# Cloud Run MCP 學習教材

## 概述

本教材旨在協助開發者了解如何利用 **Cloud Run MCP Server** 透過 AI Agent 直接管理與部署 Google Cloud Run 服務。透過此工具，您可以將雲端操作深度整合至開發工作流中。

## 文件目錄

| 文件                                                         | 說明                                               |
| ------------------------------------------------------------ | -------------------------------------------------- |
| [01--introduction.md](./01--introduction.md)                 | 介紹 Cloud Run MCP 的基本概念與環境準備。          |
| [02--core-capabilities.md](./02--core-capabilities.md)       | 詳解查詢服務、檢視狀態與 Log 分析的核心功能。      |
| [03--deployment-workflows.md](./03--deployment-workflows.md) | 說明三種不同的部署策略：映像檔、檔案夾與單檔部署。 |

## 學習路徑建議

1. 先閱讀 [01--introduction.md](./01--introduction.md) 確保您的環境與權限設定正確。
2. 嘗試 [02--core-capabilities.md](./02--core-capabilities.md) 中的指令來盤點現有資源。
3. 參考 [03--deployment-workflows.md](./03--deployment-workflows.md) 實際進行一次部署測試。
