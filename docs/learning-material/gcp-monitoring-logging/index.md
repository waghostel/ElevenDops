# GCP Monitoring & Logging 學習教材

## 概述

本教材介紹 Google Cloud 的監控與日誌服務。涵蓋 Cloud Logging 的基礎使用、Python 結構化日誌的最佳實務，以及 Error Reporting 與警報設定。

## 文件目錄

| 文件                                                                   | 說明                                         |
| ---------------------------------------------------------------------- | -------------------------------------------- |
| [01--cloud-logging-basics.md](./01--cloud-logging-basics.md)           | Cloud Logging 的核心概念與控制台操作。       |
| [02--structured-logging-python.md](./02--structured-logging-python.md) | 在 Python 中實作結構化日誌，方便查詢與分析。 |
| [03--error-reporting-alerts.md](./03--error-reporting-alerts.md)       | 自動偵測錯誤並設定警報通知。                 |

## 學習路徑建議

1. 首先閱讀 [01--cloud-logging-basics.md](./01--cloud-logging-basics.md) 了解 Cloud Logging 概念。
2. 接著參考 [02--structured-logging-python.md](./02--structured-logging-python.md) 改善程式碼的日誌輸出。
3. 部署前閱讀 [03--error-reporting-alerts.md](./03--error-reporting-alerts.md) 設定監控警報。
