# Usage Analytics: System Performance Monitoring

## Overview

The Agents Dashboard provides real-time insights into your ElevenLabs agents. It displays performance metrics over customizable time periods.

## Analytics Metrics

You can monitor activity over various daily, weekly, and monthly time periods. The dashboard includes:

- **Number of calls**
- **Average duration**
- **Total cost**
- **Average cost**

## Language Breakdown

The Language Breakdown section shows the percentage of calls (overall, or per-agent) in each language.
